# Anabel in Sonic Classic Games

Welcome to the official repository for **Anabel in Sonic Classic Games**, a comprehensive modding project bringing Anabel as a brand-new playable character to 12 iconic Sonic the Hedgehog classic games, ROM hacks, and modern engines.

## 🎮 Supported Games & Engines

This project aims to provide a unique gameplay experience for Anabel across the following titles:

| Game / Engine | Release Year | Mod Type |
| :--- | :--- | :--- |
| **Sonic 1** | 1991 | Genesis ROM Hack |
| **Sonic 1 Classic** | 2013 | RSDK v4 (Mobile/Decomp) |
| **Sonic 1 Forever** | 2024 | Community Decomp Enhancement |
| **Sonic CD** | 1993 | Sega CD ROM Hack |
| **Sonic CD Classic** | 2011 | RSDK v3 (PC/Mobile/Decomp) |
| **Sonic CD Miracle Edition** | 2023 | Community Overhaul Mod |

## 📂 Project Structure

The repository is organized into specific directories for each game version to ensure compatibility with different modding tools and engines:

- `/Sonic1_Genesis/`: Assets for the original 1991 ROM hack.
- `/Sonic1_Forever/`: Scripts and sprites for the S1F engine.
- `/SonicCD_Miracle/`: Assets for the Miracle Edition overhaul.
- `/Sonic2_Absolute/`: Scripts and sprites for the S2A engine.
- `/Sonic3_AIR/`: JSON scripts and sprite sheets for Sonic 3 A.I.R.
- `/Sonic_Mania/`: RSDK v5 mod files for Sonic Mania Plus.
- `/Shared_Assets/`: High-resolution character art, palettes, and design documents.

## 🛠️ Technical Requirements

Each engine has specific requirements for custom characters:

### **Retro Engine (RSDK) Mods**
*Used for Sonic 1/2 Classic, Sonic CD 2011, Sonic 1 Forever, and Sonic 2 Absolute.*
- **Tools:** RSDK Animation Editor, SpriteSheet tools.
- **Format:** `.gif` or `.png` sprite sheets with specific indexed palettes.

### **Sonic 3 A.I.R.**
- **Tools:** Any image editor (Aseprite recommended), JSON editor.
- **Format:** Character metadata is handled via `mod.json` and custom script files.

### **Genesis ROM Hacks**
- **Tools:** SonLVL, Hivebrain/ASMNK Disassemblies.
- **Format:** 4-bpp tiled graphics.
